# 个人介绍站 — React + Vite 实施方案

## 一、项目概述

基于你的工业工程背景与制造业经验，打造一个**简约、稳重、专业**的个人介绍站。整站为单页（SPA），共 5 个整屏/近整屏模块，版心 ≤ 1700 px，PC 端优先。

---

## 二、设计风格定义

| 维度 | 方案 |
|------|------|
| 色调 | 深蓝灰主色 `#1B2838`，钢铁灰辅色 `#4F535E`，亮白文字 `#F5F6F8`，工业蓝点缀 `#3B82F6` |
| 字体 | 英文 `Inter`（Google Fonts），中文 `Noto Sans SC`；大标题粗体 700，正文 400 |
| 质感 | 微磨砂背景、细线分隔、低饱和渐变、卡片投影 `0 4px 24px rgba(0,0,0,0.12)` |
| 动效 | 滚动渐入（Intersection Observer）、数字递增、hover 微位移，**不做花哨粒子特效** |
| 图标 | `react-icons`（Lucide 或 Heroicons 风格），线性图标，统一 stroke 宽度 |

> [!NOTE]
> 整体视觉对标制造业集团官网风格：大留白、粗字重、冷色调、数据可视化感强。避免科技创业公司的多彩渐变。

## 三、技术栈与项目结构

### 技术选型

| 技术 | 用途 |
|------|------|
| **Vite 6 + React 19** | 构建与框架 |
| **Vanilla CSS + CSS Modules** | 样式隔离，无 Tailwind |
| **react-icons** | 图标库 |
| **Intersection Observer API** | 滚动动画（原生，不引入 AOS 等库） |
| **react-countup** | 数字递增动效（项目数据展示） |

## 五、关键交互与动效

| 效果 | 实现方式 |
|------|----------|
| 滚动渐入 | 自定义 `useScrollReveal` hook，基于 Intersection Observer，元素从下方 30px 淡入 |
| 数字递增 | Profile 模块数据卡片进入视口时触发 CountUp |
| 导航高亮 | Intersection Observer 检测当前可见 section，高亮对应导航项 |
| 视频背景 | `<video>` 标签 autoplay / muted / loop / playsInline，叠加渐变遮罩 |
| 卡片 Hover | 项目卡片 `transform: translateY(-6px)` + `box-shadow` 加深，过渡 0.3s |
| 平滑滚动 | CSS `scroll-behavior: smooth`，导航锚点跳转 |

## 六、初始化与构建步骤

```bash
# 1. 查看 create-vite 可用选项
npx -y create-vite@latest --help

# 2. 在 简历/personal-site/ 目录下初始化 Vite + React 项目
npx -y create-vite@latest ./personal-site --template react

# 3. 安装依赖
cd personal-site && npm install

# 4. 安装额外依赖
npm install react-icons react-countup

# 5. 启动开发服务器
npm run dev
```

## 八、验证计划

- `npm run dev` 启动后检查每个模块渲染是否正常
- 浏览器 DevTools 检查版心宽度是否 ≤ 1700 px
- 检查滚动动画、数字递增、导航高亮等交互效果
- 检查视频背景加载与遮罩叠加效果

### 构建验证
- `npm run build` 确认无编译错误
- `npm run preview` 检查生产版本表现
